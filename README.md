## TradeSim
Practice Trading Without Risk

## About
TradeSim is a MERN-based stock trading simulator web application that allows users to practice stock trading in a virtual environment without financial risk.

Users can:
1. Search live stocks
2. View stock charts and analytics
3. Buy and sell stocks
4. Track portfolio performance
5. Monitor profit/loss
6. View financial news

The platform helps beginners understand stock market trading through real-time simulation.

## Install Dependencies
Open terminal:
1. npm install
2. npm run install-client
   
Run Backend
1. cd server
2. npm run start

Run Frontend
1. cd client
2. npm start
